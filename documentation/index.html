createDOMBlocks

TypeScript könyvtár komplex HTML blokkok és űrlapelemek létrehozásához.

Ideális választás, ha egy egyszerű, de dinamikus oldalt kell létrehoznod egyetlen HTML fájlban — bundler, frontend keretrendszer vagy build eszközök nélkül. Elég egyetlen CDN sor, és máris használhatod a több mint 50 komponenst.
