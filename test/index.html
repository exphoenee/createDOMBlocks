Telepítés

npm

npm install createdomblocks

CDN (unpkg)

<link rel="stylesheet" href="https://unpkg.com/createdomblocks/dist/style.css" />
<script src="https://unpkg.com/createdomblocks/dist/index.js"></script>

Ez egy globális CreateDOMBlocks objektumot hoz létre az összes exportált függvénynyel.

Gyors használat

npm-mel

import { createTextInput, createButton, createTable } from "createdomblocks";
import "createdomblocks/style.css";

const parent = document.getElementById("app");

// Szöveg input létrehozása
createTextInput({
  parent,
  id: "nev",
  labelText: "Név:",
  placeholder: "Adja meg a nevet",
});

// Gomb létrehozása
createButton({
  parent,
  id: "submitBtn",
  text: "Küldés",
  click: () => console.log("Kattintás!"),
});

Típusok

Minden komponens teljes TypeScript típusbiztonsággal rendelkezik. Az összes típus importálható a könyvtárból:

import type {
  BaseInputParams,
  TextInputParams,
  ButtonParams,
  TableParams,
  ModalParams,
  // ... további típusok
} from "createdomblocks";

Komponens katalógus

Minden komponens látható a teszt oldalakon kódpéldákkal és a tényleges eredmény.

Forms 18 + 6 egyedi

Alapértelmezett ürlap inputok és egyedi kiválasztók. Szöveg, telefon, URL, keresés, email, jelszó, szám, dátum, idő, hónap, hét, jelölő négyzet, szín, fájl, csúska, rejtett input, valamint custom date/week/month/datetime/range picker és drag-and-drop fájl feltöltő.

Form Inputok 18 Egyedi Inputok 6

Buttons 4

Gomb komponensek: button input, submit, reset és altalános <button> elem stylsal, szöveggel és kattintás kezelővel.

Gombok 4

Selection 4

Kiválasztás komponensek: natív select, radio gomb csoport, egyedi legördülő menü (createCustomSelect) és szövegdoboz.

Select / Radio / CustomSelect 4

Tables 2

Táblázatok tömböl vagy objektumból építve, fejléccel, lábléccel, sor számokkal és összeg számítással.

Táblázatok 2

Lists 2

Rendezett és rendezetlen listák létrehozása adatokból, indulásos számmal.

Listák 2

Navigation 3

Navigációs komponensek: menü sáv almenüvel, morzsa menü és lapozható tabok.

Nav / Breadcrumb / Tabs 3

Content 11

Tartalom megjelenítő komponensek: kártya, rács, container, bekezdés, címsor, idézet, kódblokk, kép, link, elválasztó vonal, új sortörés.

Tartalom Komponensek 11

Feedback 5

Visszajelzés komponensek: alert (4 típus), toast értesítés, badge állapot jelző, spinner töltés indikátor, progress bar folyamatjelző.

Visszajelzés 5

Interactive 2

Interaktív komponensek: összecsukható szekciók (accordion) és buborék súgó (tooltip) pozícionálással.

Accordion / Tooltip 2

Modal 1

Modál dialógus ablak címmel, testtel és gombokkal. Portálként a body végére kerül, animációval nyílik és záródik.

Modal 1

Avatar 3

Felhasználó avatar képpel vagy monogrammal, meretre váltható és alakzat opciókkal (kör, négyzet).

Függőség: DOMelemJS

Ez a könyvtár a DOMelemJS könyvtárra épül, ami egy légsúlyú, nulla függőség TypeScript könyvtár HTML elemek dinamikus létrehozásához.

A createDOMBlocks minden komponense a createDOMElem() függvényt használja a DOM elemek közvetlen létrehozásához — nincsenek HTML string sablonok.
