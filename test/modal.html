Modal

createModal

Modál dialógus ablak, ami portálként a body végére kerül. Animációval nyílik és záródik.

const modal = createModal(
  { modalTitle: "Pelda Modal", body: { tag: "p", text: "Ez egy modal tartalma." } },
  { okAction: () => console.log("OK"), cancelAction: () => console.log("Megse") },
  { id: "demoModal" }
);

createButton({
  parent: "#app",
  text: "Modal megnyitasa",
  click: () => modal.__openModal(),
});
