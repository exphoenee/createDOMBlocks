Interaktív Komponensek

createAccordion

Összecsukható szekciók címekkel és tartalommal. Több szekció is nyitva lehet.

createAccordion({
  parent: "#app",
  id: "accordion1",
  items: [
    { id: "acc1", title: "Elsó szekció",
      content: { tag: "p", text: "Elsó szekció tartalma." }, open: true },
    { id: "acc2", title: "Második szekció",
      content: { tag: "p", text: "Második szekció tartalma." } },
    { id: "acc3", title: "Harmadik szekció",
      content: { tag: "p", text: "Harmadik szekció tartalma." } },
  ],
});
