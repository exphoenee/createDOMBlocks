createCustomSelect

Custom legördülő egyéni stílussal, külön click handlerrel változtatásnál.

createCustomSelect({
  parent: "#app",
  id: "customSelect",
  labelText: "Custom select:",
  placeholder: "Valassz opciot...",
  options: [
    { text: "Budapest", value: "bp" },
    { text: "Debrecen", value: "dc" },
    { text: "Szeged", value: "sz" },
  ],
  onChange: (val) => console.log("Custom select:", val),
});
