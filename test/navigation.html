Navigáció

createNav

Navigációs sáv menüpontokkal és legördülő almenüvel.

createNav({
  parent: "#app",
  id: "nav1",
  items: [
    { text: "Fooldal", href: "#", active: true },
    { text: "Rolunk", href: "#about" },
    { text: "Szolgaltatasok", href: "#services",
      children: [
        { text: "Webfejlesztes", href: "#" },
        { text: "Mobil app", href: "#" },
      ]
    },
    { text: "Kapcsolat", href: "#contact" },
  ],
});

createTabs

Lapozható tabok külön tartalommal.

createTabs({
  parent: "#app",
  id: "tabs1",
  tabs: [
    { id: "tab1", label: "Beallitasok",
      content: { tag: "p", text: "Beallitasok tartalma" } },
    { id: "tab2", label: "Profil",
      content: { tag: "p", text: "Profil tartalma" } },
    { id: "tab3", label: "Ertesitesek",
      content: { tag: "p", text: "Ertesitesek tartalma" } },
  ],
});
