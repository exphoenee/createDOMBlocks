Visszajelzés Komponensek

createAlert

Figyelmeztető üzenet típusonként: success, error, warning, info. Elcsukható (dismissible) opcióval.

createAlert({ parent: "#app", type: "success", title: "Siker!", message: "A mentes sikeres volt." });
createAlert({ parent: "#app", type: "error", title: "Hiba!", message: "Valami rosszul sikerult." });
createAlert({ parent: "#app", type: "warning", title: "Figyelem!", message: "Kosd be a biztonsagi ovet." });
createAlert({ parent: "#app", type: "info", title: "Info", message: "Ez elcsukhato.", dismissible: true });

createBadge

Állapot jelző pici szöveggel.

createBadge({ parent: "#app", text: "Uj", type: "info" });
createBadge({ parent: "#app", text: "Siker", type: "success" });
createBadge({ parent: "#app", text: "Hiba", type: "error" });
createBadge({ parent: "#app", text: "Figyelmeztetes", type: "warning" });
createBadge({ parent: "#app", text: "Semleges", type: "neutral" });
Eredmény:

createSpinner

Töltés indikátor meretre váltással (sm, md, lg).

createSpinner({ parent: "#app", size: "sm" });
createSpinner({ parent: "#app", size: "md" });
createSpinner({ parent: "#app", size: "lg" });
Eredmény:

createProgressBar

Folyamatjelző egyéni színnel és százalék kijelzéssel.

createProgressBar({ parent: "#app", value: 75, label: "Letoltes:", showPercentage: true });
createProgressBar({ parent: "#app", value: 30, color: "#22c55e", label: "Telepites:", showPercentage: true });
Eredmény:

createToast

Ideiglenes értesítés, automatikusan eltünik.

createToast({ message: "Sikeres mentes!", type: "success", duration: 3000 });
createToast({ message: "Hiba tortent!", type: "error", duration: 3000 });
createToast({ message: "Figyelmeztetes!", type: "warning", duration: 3000 });
